HTML Reference Page

Element: HTML
Tag Used: <html>- in beginning of the document
                 </html>- in end of the whole document
About: The root element of an html page
Appearance: Should not appear in actual browser

Element: Header
Tag Used: <head>- placed right after html tag
                 </head>- placed after defining the head (usually title)
About: Contains meta information about the document
Appearance: Should not appear in actual browser

Element: Title
Tag Used: <title>- placed after head tag
                 </title>- placed after defining title
About: specifies a title for the document
Appearance: Should not appear in actual browser

Example format:
<html>
<head>
<title>Page Title</title>
</head>
</html>

Element: Body
Tag Used: <body>- when starting to actually write content
                 </body>- after done writing the content
About: contains the visible page content
Appearance: Should not appear in actual browser, but is needed in order for content to appear

Element: Headings
Tag Used: <h1>,<h2>... and </h1></h2>...
About: <h1> defines a large heading, <h2> would be a smaller heading, <h2> would be even smaller and so on.
Appearance:

Heading 1


Heading 2


Element: Paragraphs
Tag Used:<p> and </p>
About: defines a paragraph
Appearance: "This is a paragraph"

Element: Breaks
Tag Used: <br>
About: defines a line break, an empty element without a closing tag
Appearance: This is
a line break

Element: Lines
Tag Used: <hr>
About: a line going across the page, used when dividing sections

Element: Comments
Tag Used: <!-- -->
About:Can place notifications and reminders in your HTML or can be used to debug html
Appearance: Should not appear in actual broswer
