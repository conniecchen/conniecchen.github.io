Element: HTML

Syntax:

<html>- in beginning of the document
</html>- in end of the whole document

About:

The root element of an html page

Example:

<html>
.....
</html>

Appearance:

Should not appear in actual browser
W3 Schools

Element: Head

Syntax:

<head>- placed right after html tag
</head>- placed after defining the head (usually title)

About:

Contains meta information about the document

Example:

<html>
<head>
.....
</head>
</html>

Appearance:

Should not appear in actual browser
W3 Schools

Element: Title

Syntax:

<title>- placed after head tag
</title>- placed after defining title

About:

Specifies a title for the document

Example:

<html>
<head>
<title>Page Title</title>
</head>
</html>

Appearance:

Should not appear in actual browser
W3 Schools

Element: Body

Syntax:

<body>- when starting to actually write content
</body>- after done writing the content

About:

Contains the visible page content

Example:

<body>
.....
</body>

Appearance:

Should not appear in actual browser, but is needed in order for content to appear
W3 Schools

Element: Headings

Syntax:

<h1>Heading 1</h1>, <h2>Heading 2</h2>, etc

About:

<h1> defines a large heading, <h2> would be a smaller heading, <h2> would be even smaller and so on.

Example:

<body>
<h1>Heading1 </h1>
<h2>Heading 2</h2>
</body>

Appearance:


Heading 1


Heading 2


W3 Schools

Element: Paragraphs

Syntax:

<p>Content</p>

About:

defines a paragraph

Example:

<body>
<p>This is a paragraph.</p>
</body>

Appearance:

This is a paragraph
W3 Schools

Element: Breaks

Syntax:

<br>

About:

defines a line break, an empty element without a closing tag

Example:

<p>This is <br> a line break </p>

Appearance:

This is
a line break

Syntax:

a line going across the page, used when dividing sections

Syntax:

About:

Element: Links

Syntax:

<a> (popular one is <a href>) and </a>

About:

The href attribute specifies the destination address. The link text is the visible part.

Example:

<a href="url">link text</a>

Appearance:

Exercise Home

W3 Schools

Element: Images

Syntax:

<img>

About:

The <img> tag is empty and does not have a closing tag. The src attribute defines the URL of the image.

Example:

<img src="url">

Appearance:



W3 Schools

Element: Lists

Syntax:

<ul>-starts an unordered list (just bullet point)
<ol>-starts an ordered list(numbered list)
<li>- starts each list item

About:

Used to list items (numbered, bullets,letters, etc)

Example:

<ul>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>

Appearance:

  • Coffee
  • Tea
  • Milk

W3 Schools

Element: Tables

Syntax:

<table>- defines a table
<tr>- defines a table row
<th>- defines a table heading
<td>- defines the data in table

About:

used to create a table

Example:

<table style="width:300px">
<caption>Names</caption>
<tr>
  <th>Course Name</th>
  <th>Teacher</th>
  <th>Period</th>
</tr>
<tr>
  <td>Math</td>
  <td>Johnson</td>
  <td>1</td>
</tr>
<tr>
  <td>English</td>
  <td>Poppy</td>
  <td>2</td>
</tr>
<tr>
  <td>Chemistry</td>
  <td>Burke</td>
  <td>3</td>
</tr>
</table>

Appearance:

Course Name Teacher Period
Math Johnson 1
English Poppy 2
Chemistry Burke 3
