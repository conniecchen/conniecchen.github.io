Element: HTML
Tag Used: <html>- in beginning of the document
                 </html>- in end of the whole document
About: The root element of an html page
Appearance: Should not appear in actual browser

Element: Head
Tag Used: <head>- placed right after html tag
                 </head>- placed after defining the head (usually title)
About: Contains meta information about the document
Appearance: Should not appear in actual browser
Element: Title
Tag Used: <title>- placed after head tag
                 </title>- placed after defining title
About: specifies a title for the document
Appearance: Should not appear in actual browser

Example format:
<html>
<head>
<title>Page Title</title>
</head>
</html>
Element: Body
Tag Used: <body>- when starting to actually write content
                </body>- after done writing the content
About: contains the visible page content
Appearance: Should not appear in actual browser, but is needed in order for content to appear
Element: Headings
Tag Used: <h1>Heading 1</h1>, <h2>Heading 2</h2>
, etc About: <h1> defines a large heading, <h2> would be a smaller heading, <h2> would be even smaller and so on.
Appearance:

Heading 1


Heading 2

Element: Paragraphs
Tag Used:<p>Content</p>
About: defines a paragraph
Appearance: "This is a paragraph"
Element: Breaks
Tag Used: <br>
About: defines a line break, an empty element without a closing tag
Appearance: This is
a line break
Element: Lines
Tag Used: <hr>
About: a line going across the page, used when dividing sections
Appearance: is the line below
Element: Comments
Tag Used: <!-- -->
About: Can place notifications and reminders in your HTML or can be used to debug html
Appearance: Should not appear in actual broswer
Element: Links
Tag Used: <a> (popular one is <a href>) and </a>
Example: <a href="url">link text</a>
About: The href attribute specifies the destination address. The link text is the visible part.
Appearance: Exercise Home
Element: Images
Tag Used: <img>
Example: <img src="url">
About: The <img> tag is empty and does not have a closing tag. The src attribute defines the URL of the image.
Appearance:
Element: Lists
Tag Used: <ul>-starts an unordered list (just bullet points)
                 <ol>-starts an ordered list(numbered list)               
                 <li>- each list item starts with this
Example:
<ul>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>
About: Used to list items (numbered, bullets,letters, etc)
Appearance:
  • Coffee
  • Tea
  • Milk
Element: Tables
Tag Used: <table>- defines a table
                <tr>- defines a table row
                <th>- defines a table heading
               
                <td>- defines the data in table
Example:
<table style="width:300px">
<caption>Names</caption>
<tr>
  <th>Course Name</th>
  <th>Teacher</th>
  <th>Period</th>

</tr>

</tr>

Element: Blocks
Tag Used: <div>
About: to define a section of a document, used when wanting to specify a whole section a certain style. For example, below, I specified the whole section as the background color black and text white.
